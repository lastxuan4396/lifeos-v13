PLAYBOOK ENGINE

LifeOS v1.3 Daily Engine

单入口：只填 DL-30，自动生成 A/B/C/D。你只执行 B。

数据策略：默认仅保存在当前浏览器（localStorage），不会自动上传服务器。可在“数据与迁移”里导出/导入。

30秒上手（先看这个）
  1. 左侧只填 5 项事实：行为 / 结果 / 昨日B执行 / RUIN / 明日意图。
  2. 提交后右侧生成 A/B/C/D。
  3. 你只执行 B：普通日 <=15 分钟，红灯日 <=5 分钟。
  4. 次日只回答“昨日B执行 0/1”，系统自动闭环。

原则：不断更 > 完美；红灯先防崩，再谈推进。

术语速查（输入5词 + 输出5词）

输入区看这里

  • DL-30：每日最小记录，只写事实。
  • 昨日B执行 0/1：昨天系统B做没做。
  • RUIN 0/1：是否进入红灯防崩模式。
  • S / T / P：S睡眠红灯，T交易违规，P关键任务掉线。
  • 明日意图：一句方向；留空走默认B。

输出区看这里

  • A 模式判断：稳态 / 重启 / 红灯。
  • B 最小行动：唯一发动机；你只执行这一步。
  • C 风险+防线：最易翻车点 + 默认防线。
  • D tail bet：可选上行，不做不影响闭环。
  • Scoreboard-5：看趋势，不看一时情绪。
